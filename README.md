# Mets-2026-Schedule
Static web page that gives an overview of the Mets 2026 MLB season.


Full 2026 schedule for the New York Mets via the Major League Baseball Stats API

*   Opponent logos for all 30 teams

*   Home vs away shading

*   Clickable → MLB GameDay

*   Hover tooltip showing
    *   probable pitchers
    *   game time

Could update line
```
const TEAM_ID=121;
```
